# WQ_JS_Libraries
BANFF ESCAPE - TRAVEL / VACATION WEB PAGE

Files included:
- index.html
- style.css
- script.js

Third-party JavaScript libraries used:
1. Glide.js
2. AOS (Animate On Scroll)
3. Leaflet
4. Lightbox2

Notes:
- This project is built mobile-first and becomes more complex at larger screen sizes.
- If your browser blocks local assets when opening the HTML file directly, run it with Live Server or another local dev server.
- All third-party libraries are loaded through CDN links in index.html.
